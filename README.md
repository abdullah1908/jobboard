## Plugin Features ##
* Job Types
* Job Locations
* Job Specialisms
* Job Tags
* Jobs Listing
* Companies Listing
* Candidates Listing
* Employer Dashboard
* Candidate Dashboard
* Manage Profile
* Application History
* Front-End Add Job
* Front-End Login Form
* Employer Role
* Candidate Role
* Job Manager Role
* RC Framework
* Many More Features
